Basée sur le guide ITSAP.10.035 du CCCS, cette matrice de correspondance aligne les Contrôles de cybersécurité de base avec la norme ISO/IEC 27001:2022, structurée pour la plateforme CISO Assistant. Ce formatage permet l'importation de la cartographie via le standard NIST OLIR pour une gestion automatisée de la conformité. 

Pour plus de détails sur la méthode de mapping, consultez la documentation CISO Assistant (https://intuitem.gitbook.io/ciso-assistant/guide/installation).
