Basée sur le guide ITSAP.10.035 du CCCS, cette matrice de correspondance aligne les Contrôles de cybersécurité de base avec la norme ISO/IEC 27001:2022, structurée pour la plateforme CISO Assistant. Ce formatage permet l'importation de la cartographie via le standard NIST OLIR pour une gestion automatisée de la conformité. 

Pour plus de détails sur la méthode de mapping, consultez la documentation CISO Assistant (https://intuitem.gitbook.io/ciso-assistant/guide/installation).

Add ITSAP.10.035 IT Security Guideline
Cette PR ajoute le référentiel ITSAP.10.035 une ligne directrice de sécurité des TI publiée par le Centre canadien pour la cybersécurité. Ce document fournit des recommandations pratiques pour la protection des systèmes et des données, notamment en matière de configuration sécurisée, de gestion des accès et de surveillance.

Source
Transcrit à partir du document officiel ITSAP.10.035 du Centre canadien pour la cybersécurité (cyber.gc.ca).

Structure
13 catégories de contrôle
16 sous-catégories
57 exigences ou mesures de sécurité (évaluables), avec les sous-points intégrés dans la description
La numérotation suit celle du document ITSAP.10.035 afin de conserver la correspondance avec la source officielle.

Note pour les reviewers
Le document source contient plusieurs niveaux de listes imbriquées (numérotation, lettres, sous-points). Comme pour les autres référentiels, seuls les éléments principaux sont modélisés comme exigences évaluables. Les sous-points sont intégrés dans la description de chaque exigence, avec leur structure conservée (ex. : (a), (1), etc.), afin de préserver l’intégralité du contenu tout en maintenant une structure exploitable.
